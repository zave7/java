상속 (Inheritacne)

1. 상위 class 의 field 와 method 를 하위 class 에서 객체 생성 없이 사용
	단, private인 field와 method는 상속 x
	생성자 상속 x
2. extends keyword 사용
	형식)
	접근제어자 class ClassName(하위 클래스) extends ClassName(상위 클래스) {	
	}
3. 클래스 간의 상속은 단일 상속만 가능하다.
	다중상속 불가
	계층적 상속은 가능
4. 모든 클래스의 최상위 클래스는 Object 이다.
5. method overriding(재정의) : 하위 클래스의 기능에 맞게끔 재정의
	method 의 선언은 그대로... (단, 접근제어자는 넓은 의미로 변경 가능)
6. this
   super : 상위 클래스 참조 변수
   this()
   super() : 상위 클래스의 생성자 호출

7. 다형성 : 하위 클래스를 참조하여 상위 클래스의 객체를 생성 // 상속시에 상위클래스들을 각각 생성하기 때문에 
	A.class상위 B.class하위
	A a = new A();
	B b = new B();
	A a = new B();
8. class의 관계
	has a : 집합
	is a  : 상속
